# Movie For You
## AI Movie Recommendation Project
- 사용자의 단어 입력에 따른 영화를 추천해주는 인공지능 추천 시스템입니다.
- 소셜 리뷰 데이터를 기반으로 입력값과의 코사인유사도를 측정하여 영화를 추천합니다.
  - 복수개의 단어 입력 가능 (예: 박진감+유쾌한 / 가족+힐링 )
  - 추천받은 영화에 대한 감성분석 제공
  - 리뷰 데이터 기반 평가지수 제공
